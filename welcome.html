Welcome to AgentAnything V2

You have installed the autonomous browser orchestration layer. Before you begin, you must understand and accept the operational security implications.

⚠️ Privacy & Data Warning

AgentAnything acts as a bridge. When you assign a tab as a "Target", this extension will:

  • Scrape the text and interactive elements of that page.
  • Redact detected PII (Emails, Phone Numbers, Credit Cards).
  • Transmit this data to the "Agent" tab (e.g., ChatGPT, Claude, Gemini).

By proceeding, you acknowledge that page content from Target tabs will be sent to your active AI provider's servers according to their privacy policy. Do not use this tool on tabs containing sensitive personal or financial data that you do not wish to share with the AI.

How to Use

  1. Open an AI chat (ChatGPT, Claude, etc.) and click the extension icon -> MAKE AGENT.
  2. Navigate to a website you want to control and click the extension icon -> MAKE TARGET.
  3. Type a command in the Agent tab (e.g., "Find the latest news").
  4. Watch the Agent autonomously control the Target tab.
